# Graficas2_P2

# **Todo List**

* Generador de Terreno Procedural: La idea consiste en lanzar un rayo, samplear la textura 2D aplanando las coordenadas del rayo y determinar el color del fragmento.

* StoneBurner: La idea es alterar la textura en 0 los valores de densidad del radio alrededor del punto.

* Ray Marching Iluminado: Se debe usar el cross entre la fuente de luz, o cada fuente de luz, y el pundo del sample actual. Posteriormente aplicar algun modelo de iluminacion.

* Marching Cubes: La idea es generar un mallado apartir del volumen.
